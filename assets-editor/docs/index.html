A) Installation - top

This tool is a standalone tool, including HTML, CSS and Javascript files. You can upload them to any folder by FTP and access this tool by www.yourdomain.com/folder/index.html

B) Features - top

Left menu

  • Change type of shirt
  • Change color of shirt
  • Upload your image
  • Add text
  • Add image from album

Top menu

  • Move object
  • Resize object
  • Rotate object
  • Align object
  • Bring to front/Send to back
  • Flip
  • Delete

Right menu

  • Edit text
  • Select font
  • Select text color
  • Bold/Italic/Underline

Bottom menu

  • Two sides: front and back
  • Preview design
  • Download design
